The Temperature Controlled Fan project aims to develop an intelligent fan system that adjusts its speed in response to the ambient temperature. This system uses an 8051 microcontroller to monitor temperature changes and adjust fan speed accordingly. Using a temperature sensor, the system converts analog temperature readings to digital signals via an ADC (Analog-to-Digital Converter). These digital signals are then processed by the microcontroller, which adjusts the fan speed using Pulse Width Modulation. 


The primary objective of this project is to enhance energy efficiency by ensuring the fan operates at optimal speeds relative to the ambient temperature. This not only conserves energy but also provides a comfortable environment by maintaining appropriate airflow. The temperature readings are displayed on an LCD, providing real-time feedback to the user.
